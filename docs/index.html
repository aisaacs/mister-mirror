How it works

1

Plug in your iPhone

Connect via USB. Unlock it and trust this computer if it's your first time.

2

Launch Mister Mirror

He activates the screen capture subsystem — the same one QuickTime uses — in a few seconds.

3

That's it

Your iPhone screen appears in a window. Take screenshots, record video, or just watch. Mister Mirror handles the rest.

Keyboard shortcuts

Take screenshot ⌘S
Start / stop recording ⌘R
Quit ⌘Q

Made with care. Open source on GitHub.
